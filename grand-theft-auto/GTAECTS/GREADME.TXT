Grand Theft Auto - Installationsprobleme

24 Bit Version
--------------
Wenn Ihr Computer die 24 bit Version des Spieles mit defekten Grafiken
startet, wechseln Sie bitte mit dem F11 Menü den Bildschirmmodus. Dies sollte
das Problem lösen.

Die High Colour GTA Version verwendet eine implementierte Version des
"univbe" Treibers, um die 15, 16 und 32-bit Grafikmodi anzuzeigen. Wenn Ihre
Grafikkarte vom "univbe" Test nicht erkannt wird oder Sie Ihre Grafikkarte
wechseln und dies nicht automatisch erkannt wird, starten Sie "uvconfig.exe"
im "gtados" Verzeichnis. Wenn auch dies nicht zur Lösung führt, starten Sie das
24-bit GTA durch Eingabe von "gta24 -nounivbe" im "gtados" Verzeichnis. Dies
wird "univbe" übergehen und den Treiber Ihrer Grafikkarte verwenden. Alternativ
können Sie sich auch die neueste Version von "univbe" von SciTech Software
besorgen und ihn vor dem Start von GTA ausführen.

Windows Treiber ( Grand Theft Auto DirectX )
---------------
GTA DirectX kann nur ordnungsgemäß betrieben werden, wenn Sie die neuesten
Windows 95 Direct X Microsoft Treiber für Ihre Videokarte installiert haben.
Der Händler Ihrer Grafikkarte sollte Ihnen diese besorgen können.

Die aktuellen Treiber können das Spiel zusätzlich in High Colour bei geringer
Auflösung  starten (empfohlen). Eine solche Karte ist z. B. die Matrox Millenium.


Gravis Ultrasound
-----------------

Die DOS Version von GTA unter Windows95 gestartet, wird mit einer
Ultrasound keine Soundeffekte abspielen. Sie müßten hierzu Windows beenden
und im MS-DOS Modus starten bzw. die PIF-Datei von DOS-Box auf MS-DOS
Modus ändern.

DOS Version - CD Sound
----------------------
Damit Sie bei der DOS Version von GTA CD Musik hören können, müssen Sie
den MSCDEX Treiber geladen haben.  Bei der Windows 95 Version sollten Sie
diesen Treiber nicht extra starten müssen, auch wenn Sie hier die DOS Version
in einer DOS Box starten.

Sidewinder Control Pad - unter DOS GTA
-----------------------------------
Das Microsoft Sidewinder Pad ist nicht mit der DOS Version von GTA
kompatibel. Sie können es nur in der Windows (DirectX) GTA Version einsetzen.

3DFX Version
------------
3DFX GTA ist mit Voodoo 3DFX Karten kompatibel (z. B. Diamond Monster,
Orchid Righteous). Leider ist es nicht mit den neuen Rush 3DFX Karten (z. B.
Hercules Stingray ) kompatibel. Wir hoffen, dieses Problem schnell beheben zu
können, und werden dann ein Patch auf der DMA oder BMG Webseite anbieten.

Wir empfehlen Ihnen, mindestens 32MB RAM für die 3DFX GTA Version
zu besitzen.

EMM386 (DOS)
------------
Wenn Sie unter DOS 6.22 GTA spielen, in Ihrer CONFIG.SYS Datei die
Zeile DEVICE=C:\DOS\EMM386.EXE haben und die Tastatur nicht richtig
ansprechen sollte, starten Sie den Computer mit den Tasten Strg-Alt-Entfernen
neu.

Ändern Sie dann die Zeile in DEVICE=C:\WINDOWS\EMM386.EXE, um das
Problem zu lösen.
